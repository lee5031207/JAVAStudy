속성 선택자

        선택자[속성명 = 값]
        선택자[속성명 ~= 값] : 속성 의 값이 지정한 값을 단어로 포함하는 요소
        선택자[속성명 $= 값] : 속성 의 값이 지정한 값을 끝나는 요소
        선택자[속성명 ^= 값] : 속성 의 값이 지정한 값을 시작하는 요소
        선택자[속성명 *= 값] : 속성 의 값이 지정한 값을 포함하는 요소
