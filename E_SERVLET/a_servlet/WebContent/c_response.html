HttpServletResponse : 클라이언트에게 응답을 보내기 위한 기능들을 제공하는 객체
        Content-Type : 서버와 브라우저 간 통신이 발생할 때 본문으로 전달되는 데이터가 어떤 타입의 데이터
                       인지 알려주기위해 지정하는 헤더, mime-type 또는 charset 등을 작성
        MIME TYPE 
            text : text/plain, text/html, text/css, text/javascript
            image : image/gif, image/png image/jpeg
            application : apllication/octet-stream (이진데이터)
            multipart : 바운더리를 사용해 데이터 들을 구분

        URI : 인터넷상의 자원을 식별하기 위한 문자열
        URL : 인터넷상의 자원의 위치를 나타내는 문자열

        http://localhost:9090/servlet/book.html?title=자바의정석
            =>    제목이 자바의정석 인 book.html을 원한다
            =>    자원을 식별하는 문자열이기 때문에 URI

        http://localhost:9090/servlet/book.html
            =>    자원의 위치만 나타내므로 URL
